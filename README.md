# SCT_CS_01 — Caesar Cipher

A Python program to encrypt and decrypt text using the Caesar Cipher algorithm.

## Features
- Encrypt any message with a custom shift value
- Decrypt any cipher text back to the original
- Handles uppercase, lowercase, and special characters

## How to Run
```bash
python caesar_cipher.py
```

## How It Works
Each letter is shifted by N positions in the alphabet.
Example with shift 3: A → D, B → E, Z → C
